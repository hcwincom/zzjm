订单备注：

运费计算根据最终的包装总体积、总重和预先指定的快递规则计算。 若快递规则未设置请到配送管理->店铺合作快递中添加快递或给已有快递设置关联仓库，然后再联系人和付款中添加快递区域和对应的费用。 运费计算结果赋给实际运费，若用户支付运费为空，则同样赋给用户支付运费 。

用户付款=产品总金额+用户支付运费-优惠金额+税费+其它费用。 计算税费后用户付款金额会重置，产品总金额、优惠金额、税费、其它费用变动后用户付款金额也会改变 订单付款或发货后则不能修改用户付款

拆分订单后点击确认，继续拆分则会产生新订单
