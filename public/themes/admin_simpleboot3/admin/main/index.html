网站说明

排序说明

页面底部若无说明，列表页都优先按状态排序，待审核>正常>审核不通过>禁用，其次按更新时间排序，最新的更新靠前

若该页面信息没有状态，如管理员操作记录，则按更新时间排序，最新的更新靠前

,分隔符

网站内使用,作为分隔符必须是英文符号，不能是中文符号

站内信息

网站内信息分为很多种，有分类信息，产品信息，还有通知信息，公告信息。通知信息为发送给个人，公告信息为发送给所有人

输入规则

一般来说，单行输入框的文字不能超过100，简介不能超过250个字，详情（一般带编辑器,能编辑样式的）可以输入1万字

名称等常规信息不能超过50个字，排序为整数

若无特殊说明，全站统一小数位最多2位。有特殊说明的，如价格参数，4位小数
